# Scribpy User Manual

This manual is the user-facing reference for building documentation with
Scribpy. It documents the stable Python API, every CLI command, the public
result/configuration classes, and the functional pipelines executed by the
tool.

If you want to try Scribpy quickly, start with the demo project:

```bash
scribpy demo create scribpy-demo
scribpy lint --root scribpy-demo
scribpy build html --mode site --root scribpy-demo
```

Use the quick-start page when discovering Scribpy. Use the reference pages when
you need to know the exact behavior, arguments, outputs, diagnostics, or result
types of one command or function.
